# Welcome to Astro

This is a sample blog post to test the Astro setup. Astro is a modern static site builder that
delivers lightning-fast performance with a modern developer experience.

## Why Astro?

- **Performance**: Astro delivers lightning-fast performance with zero JavaScript by default.
- **Developer Experience**: Enjoy a modern developer experience with your favorite tools and
  frameworks.
- **Content-First**: Astro is designed for content-focused websites, making it perfect for blogs and
  documentation.

## Getting Started

To get started with Astro, check out the [official documentation](https://docs.astro.build).
